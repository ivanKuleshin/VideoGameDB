name: No comments or JavaDoc unless asked
description: Do not add inline comments or JavaDoc to code unless explicitly requested
type: feedback
---

Do not add inline comments or JavaDoc to any code unless the user explicitly asks for them.

**Why:** User prefers clean, self-explanatory code without unsolicited commentary.

**How to apply:** When writing or editing Java (or any) code, omit all `//`, `/* */`, and `/** */` blocks unless the user's request specifically includes comments or documentation.
