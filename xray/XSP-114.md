## Description

Verifies that when only the id field is provided in the request body, the endpoint returns 200 OK and the inserted record uses DB defaults for omitted fields (name = '', others NULL or 0).

_Requirement:_ [XSP-107](https://ivankuleshin.atlassian.net/browse/XSP-107) – AC7
