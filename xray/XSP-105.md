## Description

Verifies that requesting a non-existent ID results in 500 Internal Server Error due to `getFirst()` throwing
`NoSuchElementException` on an empty result set. This test is expected to fail until a 404 guard is implemented.

**Requirement:** XSP-98 – AC7
