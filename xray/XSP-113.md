## Description

Verifies that sending a request body with an ID that already exists in the VIDEOGAME table results in 500 Internal
Server Error due to a DB primary key constraint violation. This test is expected to fail until a duplicate ID guard is
implemented.

_Requirement:_ [XSP-107](https://ivankuleshin.atlassian.net/browse/XSP-107) – AC6
