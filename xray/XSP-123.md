# XSP-123 – [PUT /videogames/{id}] AC8 – Non-integer videoGameId is rejected by JAX-RS

## Description

Verifies that a non-integer path parameter is rejected at the JAX-RS routing layer before the method is invoked, returning 404 or 400. Requirement: XSP-115 – AC8
