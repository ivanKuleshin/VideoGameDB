# XSP-102 – [GET /videogames/{id}] AC4 – Response is valid XML when Accept: application/xml

| **Priority**   | Medium                                                                |
| **Labels**     | `automated`, `to_automate`                                            |

## Description

Verifies that the response Content-Type is `application/xml` and the body is valid XML with root element `videoGame`
when `Accept: application/xml` is set.

**Requirement:** XSP-98 – AC4
